# Yautja Translator

A web-based translator that converts English text to Yautja (Predator) language.

## Features

- English to Yautja translation
- 50+ word dictionary based on Predator lore
- Predator-themed dark/green UI
- Copy to clipboard functionality
- Mobile responsive design

## Installation

1. Download `index.html`
2. Open in any web browser
3. No dependencies or installation required

## Usage

1. Type or paste English text in the left box
2. Click **TRANSLATE** button
3. View Yautja translation in the right box
4. Click **COPY** to copy translation
5. Click **CLEAR** to reset

## Dictionary Includes

- Common phrases (hello, goodbye, thank you)
- Actions (hunt, fight, kill)
- Concepts (honor, blood, death)
- People (warrior, human, friend, enemy)
- And more...
